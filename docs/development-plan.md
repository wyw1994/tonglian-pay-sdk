# 通联支付SDK开发计划

## 项目概述
本项目是基于通联支付平台的PHP SDK，用于简化支付相关接口的调用。目前已实现统一支付和小程序链接获取功能。

## 已完成功能
1. 统一支付接口 (unifiedOrder)
2. 小程序链接获取 (appletUrl)
